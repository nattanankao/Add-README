## โครงการหุ่นยนต์กู้ภัย

## คำอธิบาย
โปรเจกต์นี้สาธิตการสืบทอดแบบหลายทางในภาษา Java โดยใช้อินเทอร์เฟซและการประกอบ (composition)

## คุณสมบัติ
- ความสามารถในการบิน
- ความสามารถในการขับขี่
- การตรวจจับความร้อน
- โมดูลการสื่อสาร

## วิธีการเรียกใช้งาน
1. Compile:
javac *.java
2. Run:
java Main
